Proper inheritance in JavaScript

## Install via [npm](https://npmjs.org)

```sh
$ npm install bloodline
```

## Usage

```js
var inherit = require( 'bloodline' )
```

```js
inherit( SomeConstructor, SuperConstructor )
```
